# Instagram Mini Clone

A mini Instagram-style application built as part of a campus drive task round.  
The project demonstrates backend API design, authentication, database relationships, and a minimal frontend integration.

---

## Features Implemented

### Authentication
- User signup and login
- Password hashing using bcrypt
- JWT-based authentication

### User System
- Follow and unfollow users
- Maintain follower and following relationships

### Posts
- Create posts with image URL and caption
- Like and unlike posts
- Comment on posts

### Feed
- Personalized feed showing posts from followed users
- Sorted by latest posts

---

## Tech Stack

**Backend**
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose
- JWT Authentication

**Frontend**
- React (Vite)
- Axios
